<![CDATA[# Contributing to IntelliGrid EnergyGuard AI

Thank you for your interest in contributing! Here's how you can help.

## How to Contribute

### 1. Report Bugs
- Open an [Issue](../../issues) with a clear title and description
- Include steps to reproduce the bug
- Attach relevant logs or screenshots

### 2. Suggest Enhancements
- Open an Issue tagged with `enhancement`
- Describe the feature and its use case
- Include mockups or references if applicable

### 3. Submit Code Changes
1. **Fork** this repository
2. **Create a feature branch**: `git checkout -b feature/your-feature-name`
3. **Commit your changes**: `git commit -m "Add: brief description"`
4. **Push to your fork**: `git push origin feature/your-feature-name`
5. **Open a Pull Request** against `main`

## Code Style

- Follow **PEP 8** for Python code
- Use descriptive variable and function names
- Add docstrings to all public functions
- Keep notebooks clean — restart kernel and run all before committing

## Commit Message Format

```
<type>: <short description>

Types: Add, Fix, Update, Remove, Refactor, Docs
```

Examples:
- `Add: LOF anomaly detection module`
- `Fix: rolling window NaN handling`
- `Docs: update Quick Start guide`

## Questions?

Feel free to open a Discussion or reach out via Issues. We're happy to help!
